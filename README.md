# Quantum Mechanics in C++

A collection of quantum mechanical problems implemented in C++.

## About

This repository contains numerical implementations of fundamental quantum mechanics problems. Each problem is implemented from scratch in C++, focusing on clarity and physical insight.

## Problems

| Problem | Description |
|---|---|
| Particle in a Box | Infinite square well — energy eigenvalues and wavefunctions |

## Structure

```
qm_cpp/
└── particle_in_a_box/
    └── particle_in_a_box.cpp
```

## Build

```bash
g++ -O2 -o particle_in_a_box particle_in_a_box/particle_in_a_box.cpp
```

## Topics Covered

- Time-independent Schrödinger equation
- Bound state problems
- Numerical methods for quantum systems
